### Installation

1. Open Obsidian
2. Go to **Settings â†’ Community Plugins**
3. Search for "YouTube to Note" 
4. Click **Install** and then **Enable**

### Setup

1. Go to **Settings â†’ YouTube to Note**
2. Enter your AI provider API key (Google Gemini or Groq recommended)
3. Set your preferred output folder
4. Click the YouTube icon in the sidebar or use the command palette

### Usage

1. Copy a YouTube URL
2. Click the YouTube icon in the sidebar or use "Process YouTube Video" command
3. Paste the URL and select your preferred format
4. Click Process and your note will be created automatically

### API Keys

The plugin supports multiple AI providers:

- **Google Gemini** - Get a free API key from [Google AI Studio](https://aistudio.google.com/)
- **Groq** - Get a free API key from [Groq Cloud](https://console.groq.com/keys)
- **Ollama** - Run locally for privacy (no API key needed)
- **OpenRouter, Hugging Face** - Additional options

### Output Formats

- **Executive Summary** - Comprehensive overview with key takeaways
- **Tutorial/Guide** - Step-by-step guide from video content  
- **Brief Summary** - Quick 100-word summary
- **3C Concept** - Compress, Compile, Consolidate format
- **Transcript Note** - Cleaned up video transcript
- **Custom** - Use your own prompt
